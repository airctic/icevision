    IceVision Documentation

    The source for IceVision documentation is in the docs/ folder. Our documentation uses extended Markdown, as implemented by MkDocs.

    Building the documentation

    • Locally install the package as described here
    • From the root directory, cd into the docs/ folder and run:
      • python autogen.py
      • mkdocs serve # Starts a local webserver: localhost:8000
