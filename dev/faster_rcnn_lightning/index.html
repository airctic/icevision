    ModelAdapter

    icevision.models.torchvision.faster_rcnn.lightning.ModelAdapter(model, metrics=None)

    Lightning module specialized for faster_rcnn, with metrics support.

    The methods forward, training_step, validation_step, validation_epoch_end are already overriden.

    Arguments

    • model torch.nn.modules.module.Module: The pytorch model to use.
    • metrics Sequence[icevision.metrics.metric.Metric]: Sequence of metrics to use.

    Returns

    A LightningModule.
